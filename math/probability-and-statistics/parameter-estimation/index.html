# 点估计

设总体 XXX 的分布函数形式已知，但它的一个或多个参数未知，借助于总体 XXX 的一个样本来估计总体未知参数的值的问题称为参数的点估计问题。

# 最大似然估计

若总体 XXX 为离散型， X1,X2,⋯,XnX_1, X_2, \cdots, X_nX1​,X2​,⋯,Xn​ 是来自 XXX 的样本，对应的观察值为 x1,x2,⋯,xnx_1, x_2, \cdots, x_nx1​,x2​,⋯,xn​ ，则这组样本取到这组对应观察值的概率为

L(θ)=L(x1,x2,⋯,xn;θ)=∏i=1np(xi;θ)L(\theta) = L(x_1, x_2, \cdots, x_n; \theta) = \prod_{i = 1}^n p(x_i; \theta)L(θ)=L(x1​,x2​,⋯,xn​;θ)=i=1∏n​p(xi​;θ)

其中， θ\thetaθ 是总体 XXX 的参数， p(xi;θ)p(x_i; \theta)p(xi​;θ) 是 XXX 的概率密度函数， L(θ)L(\theta)L(θ) 称为样本的似然函数， θ\thetaθ 是未知参数。一个直观的想法是，我们已经观察到样本取到 x1,x2,⋯,xnx_1, x_2, \cdots, x_nx1​,x2​,⋯,xn​ 这组观察值，那么使这组观察值出现的概率最大的 θ\thetaθ 就是我们要求的参数估计，记为 θ^\hat{\theta}θ^ 。这种方法称为最大似然估计法。

若总体 XXX 为连续型，似然函数变为

L(θ)=L(x1,x2,⋯,xn;θ)=∏i=1nf(xi;θ)L(\theta) = L(x_1, x_2, \cdots, x_n; \theta) = \prod_{i = 1}^n f(x_i; \theta)L(θ)=L(x1​,x2​,⋯,xn​;θ)=i=1∏n​f(xi​;θ)

中解出 θ\thetaθ 即可。

最大似然估计的核心思想是求使得当前观测值出现的概率最大时对应的参数值。

# 估计量的标准

  1. 无偏性： E(θ^)=θ\mathbb{E}(\hat{\theta}) = \thetaE(θ^)=θ ，即估计量的数学期望等于真实参数值。PS：任意阶样本矩都是总体矩的无偏估计量。
  2. 有效性： 对 θ\thetaθ 的两个无偏估计量 θ^1,θ^2\hat{\theta}_1, \hat{\theta}_2θ^1​,θ^2​ ，若 D(θ^1)≤D(θ^2)D(\hat{\theta}_1) \leq D(\hat{\theta}_2)D(θ^1​)≤D(θ^2​) ，则称 θ^1\hat{\theta}_1θ^1​ 更有效。
  3. 相合性： 若 θ\thetaθ 的估计量 θ^\hat{\theta}θ^ 当样本容量 n→∞n\rightarrow \inftyn→∞ 时收敛于 θ\thetaθ ，则称 θ^\hat{\theta}θ^ 为 θ\thetaθ 的相合估计量。

# 假设检验

基本思想：对总体的参数作出假设，然后通过抽样进行统计分析，从而判断假设是否成立。弃真错误的概率为 α\alphaα 。

提出无效假设 / 反假设（Null Hypothesis, H0H_0H0​ ）和备择假设（Alternative Hypothesis, H1H_1H1​ ），单边（ θ=θ0\theta = \theta_0θ=θ0​ ）还是双边（ θ<θ0\theta < \theta_0θ<θ0​ 或 θ>θ0\theta > \theta_0θ>θ0​ ），选定显著性水平 α\alphaα 。
